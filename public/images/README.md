# Band Images

This directory contains the main band images used throughout the website.

## Current Images

- `hero-image.jpg` - Main hero/background image for the homepage

## Adding New Images

Simply drop your high-quality band photos here. Recommended specifications:

- **Hero Images**: 1920x1080px or higher, landscape orientation
- **Format**: JPG, PNG, or WebP
- **File Size**: Optimize to under 500KB for web performance

## Usage in Code

Reference images from this directory using `/images/filename.jpg` in your components:

```tsx
<Image src="/images/hero-image.jpg" alt="Band photo" />
```
